# React Categorized Todo List
A dynamic task management application built with **React** and **Vite** that allows users to organize tasks under custom, expandable headings.

## Key Features
- Dynamic Categorization: Users can create custom headings (e.g., "Work", "Personal") to group tasks.
- Nested State Management: Uses React's `useState` hook to manage a complex data structure of arrays and objects.
- Interactive UI: Fully responsive design with real-time updates for adding and deleting items.

## Tech Stack
- Frontend: React.js
- Build Tool: Vite
- Styling: CSS3
- Logic: JavaScript (ES6+)

## How to Run Locally
1. Clone the repository.
2. Run `npm install` to install dependencies.
